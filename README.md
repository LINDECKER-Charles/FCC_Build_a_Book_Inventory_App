# 📚 Build a Book Inventory App

## 📖 Overview

This project is part of the FreeCodeCamp curriculum: **Build a Book Inventory App**.

The goal of this challenge is to build a structured HTML table representing a book inventory while applying advanced CSS techniques, particularly **attribute selectors**, descendant selectors, and pseudo-classes.

The application displays books with the following properties:

* Title
* Author
* Category
* Reading Status
* Personal Rating

The project focuses on semantic HTML structure and precise CSS selector usage rather than JavaScript functionality.

---

## 🧠 Learning Objectives

Through this project, I practiced:

* Writing semantic HTML tables (`thead`, `tbody`, `tr`, `th`, `td`)
* Structuring consistent row layouts
* Using CSS attribute selectors (`[class="..."]`, `[class^="..."]`, `[class~="..."]`)
* Applying descendant and direct child combinators
* Using pseudo-classes like `:first-child` and `:nth-child()`
* Styling UI components using linear gradients
* Improving visual hierarchy and readability

This project emphasizes understanding how CSS selectors work at a granular level.

---

## 🛠 Technologies Used

* HTML5
* CSS3

No JavaScript was required for this challenge.

---

## 🎨 Features

* Gradient-based row styling depending on reading status
* Styled status badges (Read, To Read, In Progress)
* Rating system using nested `span` elements
* Dynamic rating visualization using attribute selectors
* Modern UI enhancements (hover effects, rounded badges, improved typography)
